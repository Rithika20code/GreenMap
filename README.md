# Green Map – Digital Waste Mapping Platform

Green Map is a community-driven web application that helps citizens report waste-related issues, discover nearby recycling centers, and promote cleaner, greener communities through interactive mapping and real-time reporting.

The project was developed with the assistance of OpenAI Codex and GPT-5.6 to accelerate development through code generation, debugging, architecture planning, UI improvements, and documentation support.


# Overview

Improper waste disposal is a growing environmental challenge that affects public health and the cleanliness of our surroundings. Green Map bridges the gap between citizens and environmental action by providing an easy-to-use platform where users can report waste hotspots, visualize reported locations on an interactive map, and find nearby recycling centers.

Throughout development, OpenAI Codex and GPT-5.6 were used as AI development assistants to streamline coding, optimize project architecture, improve user experience, and generate project documentation.



# Features

- Report waste hotspots with location and description.
- Interactive map displaying reported waste locations.
- Locate nearby recycling centers.
- Report overflowing bins and illegal dumping areas.
- View clean-up drives and sustainability initiatives.
- Real-time updates of reported waste issues.
- Responsive design for desktop and mobile devices.
- AI-assisted development using OpenAI Codex and GPT-5.6.



# Objectives

- Provide a platform for reporting and tracking waste-related issues.
- Increase public awareness about improper waste disposal.
- Enable users to identify waste hotspots through an interactive map.
- Promote recycling by helping users locate nearby recycling centers.
- Encourage community participation in environmental conservation.

# Installation

## Prerequisites

- Node.js
- npm
- MongoDB

## Steps

Clone the repository.

```bash
git clone https://github.com/yourusername/green-map.git
```

Navigate to the project directory.

```bash
cd green-map
```

Install dependencies.

```bash
npm install
```

Start the development server.

```bash
npm start
```

Open the application in your browser.

```text
http://localhost:3000
```
